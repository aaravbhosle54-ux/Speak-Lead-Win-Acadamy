# Getting Started with SpeechMaster

## Step 1: Access the Website

Simply open `index.html` in your web browser. No installation or setup required!

## Step 2: Navigate the Interface

The website has 5 main sections:

1. **Home** - Overview and introduction
2. **Topic Generator** - Generate age-appropriate speech topics
3. **Outline** - Create speech outlines by type
4. **Quotes** - Find inspirational quotes
5. **Jokes** - Get family-friendly jokes

Use the navigation bar to quickly jump between sections.

## Step 3: Generate Content

### For Topics:
1. Select your age group
2. Click "Generate Topics"
3. Choose from the list or click "Copy All Topics"

### For Outlines:
1. Enter your speech title
2. Select a speech type
3. Click "Generate Outline"
4. Review the structure and copy if needed

### For Quotes:
1. Select a topic
2. Click "Get Quote"
3. Use "Get Another Quote" for variations
4. Copy the quote to clipboard

### For Jokes:
1. Click "Generate Joke"
2. Use "Get Another Joke" for more
3. Copy to use in your speech

## Tips for Best Results

### When Using Topics:
- Consider your audience's interests
- Pick topics that excite you
- Mix and match with other tools

### When Creating Outlines:
- Keep your title concise
- Match the outline type to your purpose
- Expand each point with your own research

### When Selecting Quotes:
- Choose quotes that relate to your topic
- Ensure the quote aligns with your message
- Practice delivering the quote naturally

### When Adding Jokes:
- Use jokes to break tension
- Match humor to your audience
- Ensure smooth transitions

## Mobile Use

All features work on mobile devices:
- Tap navigation items to scroll
- Use dropdown menus to select options
- Copy buttons work on all devices

## Troubleshooting

**Q: Content won't generate**
- A: Make sure you've selected all required options
- Check your browser console for errors (F12)

**Q: Copy button doesn't work**
- A: Ensure your browser has clipboard permissions
- Try a different browser if issues persist

**Q: Mobile layout looks odd**
- A: Refresh the page
- Try rotating your device
- Clear browser cache

## Keyboard Shortcuts

While the site is mouse-friendly, you can also:
- Use **Tab** to navigate between form elements
- Use **Enter** to submit forms
- Use **Esc** if needed to reset focus
