# SpeechMaster - Speech Generator Website

A comprehensive web application designed to help speakers of all ages master public speaking through instant generation of speech topics, outlines, quotes, and jokes.

## 🎤 Features

### 1. **Impromptu Topic Generator**
- Generate age-appropriate speech topics
- Supports 5 age groups:
  - Elementary School (K-5)
  - Middle School (6-8)
  - High School (9-12)
  - College/University
  - Professional/Adult (30+)
- Over 50 carefully curated topics

### 2. **Speech Outline Generator**
- Create structured outlines for different speech types
- Supported speech types:
  - **Persuasive Speech**: Convince and inspire action
  - **Informative Speech**: Educate and share knowledge
  - **Storytelling Speech**: Engage through narrative
  - **Motivational Speech**: Inspire and empower
  - **Impromptu/Toastmasters Style**: Quick, structured approach
- Each outline includes complete section breakdowns with key points

### 3. **Quote Generator**
- Access to 75+ carefully selected, non-political quotes
- 16 quote topics:
  - Public Speaking
  - Leadership
  - Integrity
  - Global Citizenship
  - Positivity
  - Energy & Enthusiasm
  - Confidence
  - Entrepreneurship
  - Teamwork
  - Growth Mindset
  - Courage
  - Gratitude
  - Discipline
  - Creativity
  - Resilience
  - Youth Leadership

### 4. **Joke Generator**
- 30+ clean, family-friendly jokes
- Perfect for engaging audiences
- Suitable for all ages and global audiences
- Free of political references and named public figures

## 🚀 Quick Start

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No backend server required - runs entirely on the client

### Installation

1. Clone the repository:
```bash
git clone https://github.com/aaravbhosle54-ux/Speak-Lead-Win-Acadamy.git
cd Speak-Lead-Win-Acadamy
```

2. Open `index.html` in your web browser:
```bash
# On macOS
open index.html

# On Windows
start index.html

# On Linux
xdg-open index.html
```

Or simply double-click `index.html` to open it in your default browser.

## 💻 Usage

### Topic Generator
1. Navigate to the "Topic Generator" section
2. Select your age group from the dropdown
3. Click "Generate Topics"
4. View and copy the generated topics

### Outline Generator
1. Go to the "Outline" section
2. Enter your speech title
3. Select the speech type
4. Click "Generate Outline"
5. Copy the complete outline for use

### Quote Generator
1. Navigate to the "Quotes" section
2. Select a quote topic
3. Click "Get Quote"
4. Click "Get Another Quote" for more options
5. Copy the quote to use in your speech

### Joke Generator
1. Go to the "Jokes" section
2. Click "Generate Joke"
3. Click "Get Another Joke" for more options
4. Copy the joke to use in your speech
